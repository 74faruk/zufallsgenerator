# Zufallsgenerator

Drei kleine Zufalls-Tools in einem: Würfel, Münzwurf und Zufallsauswahl aus
einer eigenen Liste (z. B. "Was gibt's heute zu essen?").

## Features

- Würfel: 1–6 Würfel gleichzeitig, mit Summe
- Münzwurf: mit Statistik (Kopf/Zahl-Zählung, lokal gespeichert)
- Listen-Los: eigene Optionen eingeben, zufällig eine auswählen lassen

## Tech

Reines HTML, CSS und JavaScript — kein Framework, kein Build-Schritt.

## Lokal öffnen

`index.html` einfach im Browser öffnen, fertig.
